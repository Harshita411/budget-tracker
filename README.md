# Expense Tracker (MERN Stack)

A full-stack expense tracker application built using the MERN stack
(MongoDB, Express.js, React, and Node.js).

The application allows users to securely register and log in, and then
add, view, and delete income and expense transactions. A dashboard
provides an overview of the userâ€™s financial data along with visual
charts to analyze spending and income patterns.

## Features
- User authentication using JWT
- Add and delete income records
- Add and delete expense records with categories
- Dashboard showing total balance, income, and expenses
- Charts for income and expense visualization
- Responsive user interface

## Tech Stack
- React
- Node.js
- Express.js
- MongoDB

## Purpose
This project was built to practice full-stack MERN development,
including authentication, REST APIs, database operations, and frontend
state management.
